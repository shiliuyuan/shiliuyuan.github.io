学习前景

目前Go语言已经⼴泛应用于人工智能、云计算开发、容器虚拟化、⼤数据开发、数据分析及科学计算、运维开发、爬虫开发、游戏开发等领域。

Go语言简单易学，天生支持并发，完美契合当下高并发的互联网生态。Go语言的岗位需求持续高涨，目前的Go程序员数量少，待遇好。

抓住趋势，要学会做一个领跑者而不是跟随者。

国内Go语言的需求潜力巨大，目前无论是国内大厂还是新兴互联网公司基本上都会有Go语言的岗位需求。

Go了解

Hello World

学习一门语言第一个程序都是Hello Word，接下来将使用go输出Hello World，创建hello.go文件，编辑该文件内容如下：

// 该代码仅做示范用，具体内容后续解释
package main
import "fmt"
func main(){
    fmt.Println("Hello World!")
}

运行代码：

$go run hello.go

输出：

Hello World!

或者构建程序

$go bulid hello.go

windows下会生成hello.exe可执行文件，双击即可运行，linux下则会生成二进制文件，可直接执行

go使用的是以go结尾的文件

Go的基本思想

Less can be more 大道至简,小而蕴真 让事情变得复杂很容易，让事情变得简单才难 深刻的工程文化

Go的优点

  • 静态编译，编译好后，扔服务器直接运行。
  • 简单的思想，没有继承，多态，类等。
  • 丰富的库和详细的开发文档。
  • 语法层支持并发，和拥有同步并发的channel类型，使并发开发变得非常方便。
  • 简洁的语法，提高开发效率，同时提高代码的阅读性和可维护性。
  • 超级简单的交叉编译，仅需更改环境变量。
  • 复杂度低，速度快，几乎赶上C和C++

Go 的特征

  • 自动立即回收。
  • 更丰富的内置类型。
  • 函数多返回值。
  • 错误处理。
  • 匿名函数和闭包。
  • 类型和接口。
  • 并发编程。
  • 反射。
  • 语言交互性。

Go的标识符

与其它大部分语言相同，go的标识符遵循以下规则

  • 首字符可以是任意的Unicode字符或者下划线
  • 剩余字符可以是Unicode字符、下划线、数字
  • 字符长度不限

Go变量的作用域

  1. 声明在函数内部，是函数的本地值，类似private
  2. 声明在函数外部，是对当前包可见(包内所有.go文件都可见)的全局值，类似protect
  3. 声明在函数外部且首字母大写是所有包可见的全局值，类似public

go变量声明

Go的变量声明主要有四种方式

var（声明变量）, const（声明常量）, type（声明类型） ,func（声明函数）。

引用类型

主要包括以下三种：

  • slice – 序列数组(最常用)
  • map – 映射
  • chan – 管道

Go数组Array

数组是具有相同唯一类型的一组已编号且长度固定的数据项序列，这种类型可以是任意的原始类型例如整型、字符串或者自定义类型。

声明数组

声明数组的格式如：

var balance [10] float32

初始化数组

未声明数组，初始化数组有以下三种方式

// 方式一
var balance = [5]float32{1000.0, 2.0, 3.4, 7.0, 50.0}
// 方式二
balance := [5]float32{1000.0, 2.0, 3.4, 7.0, 50.0}
// 方式三，会自行确定数组长度
var balance = [...]float32{1000.0, 2.0, 3.4, 7.0, 50.0}
//  将索引为 1 和 3 的元素初始化
balance := [5]float32{1:2.0,3:7.0}

针对已声明数组，使用以下方式进行赋值

 balance[4] = 50.0

访问数组

与其它语言访问数组方式类似

var salary float32 = balance[9]

Go切片

切片相当于一个可变长度的数组，相比数组更为灵活。其具体声明如下：

slice1 := make([]float32, 0) // 长度为0的切片
slice2 := make([]float32, 3, 5) // [0 0 0] 长度为3容量为5的切片
fmt.Println(len(slice2), cap(slice2)) // 3 5 获取切片的长度和容量

其中长度表示切片允许访问元素的个数，容量表示切片允许增长到的元素个数。

使用切片示例代码如下：

// 添加元素，切片容量可以根据需要自动扩展
slice2 = append(slice2, 1, 2, 3, 4) // [0, 0, 0, 1, 2, 3, 4]
fmt.Println(len(slice2), cap(slice2)) // 7 12
// 子切片 [start, end)
sub1 := slice2[3:] // [1 2 3 4]
sub2 := slice2[:3] // [0 0 0]
sub3 := slice2[1:4] // [0 0 1]
// 合并切片
combined := append(sub1, sub2...) // [1, 2, 3, 4, 0, 0, 0]

由上述代码可以发现，切片创建时候会给切片分配一个容量，但这个容量是可以通过append函数扩增的。

切片可以通过 切片名…展开为切片中的元素（如sub2…）

Go映射（字典）

字典和其它语言的字典几乎相同，这里不做过多介绍。

Go信道（管道）

信道是带有类型的管道，你可以通过它用信道操作符 <- 来发送或者接收值。

// 和映射与切片一样，信道在使用前必须创建：
ch := make(chan int,2)
ch <- 1    // 将1发送至信道 ch。
v := <-ch  // 从ch 接收值并赋予 v。
其中 <-就是数据流的方向

Go指针

与C/C++相同，使用&取地址，使用*取地址中的值。

Go运算符

与大部分流行的编程语言相同，Go的运算符包括：

  • 赋值运算符
  • 算数运算符
  • 逻辑运算符
  • 位运算符
  • 关系运算符

运算符的种类、优先级等都符合常识，且和C/C++类似，这里不做过多介绍。

Go流程控制语句

Go的流程控制语句分为条件语句和循环语句。

条件语句

条件语句最经典的就是if…else语句，其基本形式为：

if 条件A {
	语句A
} else if 条件B {
    语句B
} else{
    语句C
}

其中，else if 和 else可以省略。

然后是switch语句，其语法形式如下：

switch 变量A {
    case 变量B:
    	语句B
    case 变量C:
    	语句C
    default:
    	语句D
}

其中，变量A、变量B、变量C可以说任何类型，但类型必须相同，也可以使用case 变量A、变量B、…的形式同时选择多个变量。

下面是其它语言都没有的select语句：

package main

import "fmt"

func main() {
   var c1, c2, c3 chan int
   var i1, i2 int
   select {
      // 阻塞，c1中没有值，不执行
      case i1 = <-c1:
         fmt.Printf("received ", i1, " from c1\n")
      // 阻塞，c2没有缓冲空间，不执行
      case c2 <- i2:
         fmt.Printf("sent ", i2, " to c2\n")
      // 阻塞，c3没有值
      case i3, ok := (<-c3):  // same as: i3, ok := <-c3
         if ok {
            fmt.Printf("received ", i3, " from c3\n")
         } else {
            fmt.Printf("c3 is closed\n")
         }
      // case都不执行，执行default输出no communication
      default:
         fmt.Printf("no communication\n")
   }    
}

循环语句

与其它语言不同，go不存在while语句，只有for循环语句，但for循环可以起到while循环的效果。

// for循环的第一种用法，与其它语言的for循环相同，求1到10的和
package main

import "fmt"

func main() {
        sum := 0
        for i := 0; i <= 10; i++ {
                sum += i
        }
        fmt.Println(sum)
}

下面这种方式为while循环

// 省略for循环的第一个条件和第三个条件，则相当于while循环
func main() {
        sum := 1
        for ; sum <= 10; {
                sum += sum
        }
        fmt.Println(sum)

        // 这样写也可以，更像 While 语句形式
        for sum <= 10{
                sum += sum
        }
        fmt.Println(sum)
}

下面为for循环的第三种用法，通常用于遍历数组、切片、字典、通道等

package main
import "fmt"

func main() {
    map1 := make(map[int]float32)
    map1[1] = 1.0
    map1[2] = 2.0
    map1[3] = 3.0
    map1[4] = 4.0
   
    // 读取 key 和 value
    for key, value := range map1 {
      fmt.Printf("key is: %d - value is: %f\n", key, value)
    }

    // 读取 key
    for key := range map1 {
      fmt.Printf("key is: %d\n", key)
    }

    // 读取 value
    for _, value := range map1 {
      fmt.Printf("value is: %f\n", value)
    }
}

Go函数

Go使用关键字func定义函数，其定义方式如下：

func 函数名(参数名 参数类型) 返回参数类型 {
   函数体
}

示例如下（求两数之和）：

package main

import "fmt"

func sum(a int, b int) int {
	return a + b
}
func main() {
	var a int = 1
	var b int = 2
	fmt.Println(sum(a, b))
}
