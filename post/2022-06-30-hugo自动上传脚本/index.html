Hugo自动上传脚本

时间： 2022-06-30   |   分类： tools     |   阅读： 421 字 ~1分钟

本文制作了一个hugo自动上传脚本，用于个人博客的自动化上传。

首先本文是基于如下的情况下制作的

  1. 系统为windows系统

  2. windows已安装git程序，用于运行shall脚本

  3. 存在github仓库，仓库中存在hugo和master分支

  4. 本地目录结构如下：

-blog

	-hugo：博客主文件存储目录

		-.git：连接好自己github仓库的hugo分支

		-deploy.sh：脚本文件放置目录

	-master：生成站点存储目录

		-.git：连接好自己github仓库的master分支

deploy.sh文件内容如下：

#!/bin/bash

echo -e "\033[0;32mDeploying updates to GitHub...\033[0m"

# rm -rf ./public
# rm -rf ./images
python3 ./gen_readme.py
rm -rf .hugo_build.lock
rm -rf ../master/*
hugo -d ../master
cd ../master
# git init .
# Add changes to git.
git add -A

# Commit changes.
msg="rebuilding site by hugo"
if [[ $# -eq 1 ]];then 
  msg="$1"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master
if [[ $? -ne 0 ]];then
  echo "push master branch to remote failed"
  exit 1
fi
cd ../hugo
# Checkout to hugp
git checkout hugo
if [[ $? -ne 0 ]];then
  echo "checkout master failed"
  exit 1
fi
git add -A
git commit -m "$msg"
git push origin hugo
if [[ $? -ne 0 ]];then
  echo "push hugo branch to remote failed"
  exit 1
fi

# Back to master
git checkout master

在该文件所在目录打开git运行如下代码，即可生成上传代码且生成博客。

./deploy.sh
