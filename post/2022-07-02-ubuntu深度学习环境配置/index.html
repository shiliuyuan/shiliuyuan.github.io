ubuntu深度学习环境配置

时间： 2022-07-02   |   分类： tools     |   阅读： 991 字 ~2分钟

本文为记录了在配置linux服务器时候使用的各种命令及操作。

Linux服务器创建用户

  1. 进入管理员用户，一般慎用，建议只在创建用户的时候使用，更新服务器中的应用

    sudo su
sudo apt-get update

  2. 创建用户并在home文件夹下生成相应的文件夹，以下均用AAA表示用户名，并设置bash登录

    useradd -m AAA -s /bin/bash

  3. 为该用户设置密码

    passwd AAA

  4. 为该用户授权sudo权限，使其可以使用系统命令

    usermod -a -G sudo AAA

通过以上步骤，一个可以使用的用户就创建完成了，下面将描述如何在同一局域网下访问该服务器

通过同一局域网访问服务器

  1. 在服务器上使用命令ifconfig -a查看服务器的ip地址

  2. 安装openssh-server

    sudo apt-get install openssh-server

  3. 在windows下安装xshell、winscp等软件

以xshell为例，winscp同理，主机地方填入之前用ifconfig -a查到的ip地址

用户身份验证填入对应用户的账号名和密码，然后点击连接即可连接。

在用户下创建pytorch环境

可在windows下下载linux版本的anaconda，然后通过winscp传输到服务器上进行安装，anaconda可通过以下方式下载：

  1. anconda官网
  2. 如需下载老版本的anconda，也可访问清华源

下载cuda和cudnn

  1. cuda可通过Nvidia官网下载，注意下载与linux系统版本一致的cuda，也可通过清华源下载
  2. cudnn也可通过Nvidai官网下载，注意与cuda的版本对应关系

下载pytorch和torchvision，这里注意下载gpu版本的

  1. 一般访问清华源，通过此地址下载对应版本的安装包
  2. 也可以直接使用pytorch官网提供的命令安装，一般不建议这种安装方式，虽然方便，但容易出现下载困难，版本不对等问题

通过以上步骤，我们准备了anaconda、cuda、cudnn、和gpu版的pytorch、torchvision等安装包

下面进行安装，注：下面使用到的均是下载的安装包的全称，这里只简写为安装包的名字：

  1. 通过winscp将上述安装包传输到linux服务器中

  2. 安装anaconda

    bash anaconda

  3. 配置环境变量

    nano ~/.bashrc
export PATH=/home/用户名/anaconda3/bin:$PATH
source ~/.bashrc

  4. 解压cudnn的压缩包，然后将其中的lib文件夹下的文件复制到anaconda安装目录下的lib文件夹中

    tar zxvf  cudnn -C 指定路径

  5. 安装cuda

    conda install - -offline cuda

  6. 安装pytorch

    conda install - -offline pytorch

  7. 安装torchvision

    conda install - -offline torchvision
